1.Enabling Code First entry in Entity Framework
 By adding the Context class
       public DatabaseContext() : base("DefaultConnection")
        {
            Database.SetInitializer<DatabaseContext>(new CreateDatabaseIfNotExists<DatabaseContext>());
        }
        // public BlogContext(DbContextOptions<BlogContext> options) : base(options) { }

        public DbSet<BlogPost> BlogPosts { get; set; }

        public DbSet<Author> Authors { get; set; }
        }


2.Inorder to created an Database we need to do the following steps in Package manager console
 1. Enable-Migrations :-This step will create the Migration folder and Configuration class
 2.Add-Migration Initial:-This step will create the Initial Migration step.
 3.Update-Database :-This step will create the database.
